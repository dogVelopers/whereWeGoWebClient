## Getting Started

```bash
npm install # or
yarn

npm run dev # or
yarn dev
```

dependency 설치 후 실행할 수 있습니다.

API Server의 경우 json-server를 이용해 개발된 [mockup server](https://github.com/dogVelopers/whereWeGoMockServer)를 실행하여 진행할 수 있습니다.
